# RFID & NFC



The script (code.ino) can be uploaded to an Arduino Uno.

You need an Arduino Uno an RFID module (RC522) and the Grove-LCD module.

Below is the required pin layout:

| Arduino UNO | RC522 (RFID Module) | Grove-LCD |
|-------------|---------------------|-----------|
| Pin 10      | SDA                 |           |
| Pin 13      | SCK                 |           |
| Pin 11      | MOSI                |           |
| Pin 12      | MISO                |           |
| GND         | GND                 |           |
| Pin 9       | RST                 |           |
| 3,3 V       | 3,3 V               |           |
| GND         |                     | GND       |
| 5 V         |                     | VCC       |
| SDA         |                     | SDA       |
| SCL         |                     | SCL       |
